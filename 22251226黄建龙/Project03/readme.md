##### 交互方式：

WASD：移动摄像头水平位置

QE：移动摄像头竖直位置

IJKL：移动模型（Cat) 水平位置

方向键：UP， DOWN 移动 Cat 竖直位置

XYZ：Cat 分别沿着XYZ轴旋转

UO：放缩Cat
