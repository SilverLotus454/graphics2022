# 三维动画作业1-3

**实现**

- 使用glfw和glad来搭建OpenGL框架
- 实现了模型加载（obj格式）
- 实现了Phong光照模型
- 实现了相机控制，物体的旋转、平移变换
- 实现了鼠标、键盘的交互操作
- 实现了贴图功能，在一个5x5的Bezier曲面上绘制了贴图

**操作指南**

- 拖动鼠标左键实现模型的旋转，而不改变观看视角
- 拖动鼠标右键实现观看视角的改变，模型本身保持固定
- WSAD键：旋转物体本身
- 方向键：平移物体本身

**其他**

- 编译好的程序在OpenGL_2\x64\Debug\OpenGL_2.exe，可直接运行
- 具体介绍见 实验报告.docx
