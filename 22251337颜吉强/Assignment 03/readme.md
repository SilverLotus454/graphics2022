# 第三次作业

1.完成了obj文件格式的模型的读入，同时加载了漫反射贴图和高光贴图

2.加入了对多光源的支持，包含点光源、平行光源、聚光灯光源等，光源在图形中有显示

3.除了支持作业1提供的浏览模式外，按C键可以切换成一个模型为中心的浏览模式，通过wasd控制视角

4.可以运行`cmake-build-debug/shader.exe`运行程序，本程序构建于64位，同时带有assimp库dll
